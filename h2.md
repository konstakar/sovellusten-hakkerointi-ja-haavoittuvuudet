1) A01 Broken Access Control
- käyttöoikeuksia ei valvota oikein
- käyttäjä pääsee käsiksi tietoihin tai toimintoihin, joihin hänellä ei pitäisi olla oikeuksia, esim. toisen käyttäjän tietojen katselu/muokkaus ja admin-tietoihin pääseminen ilman admin-oikeuksia
- hyökkäykset usein helppoja, esim. URL-muutokset (parametrien manipulointi)
- voi johtaa tietojen muokkaukseen, poistoon tai jopa niiden vuotoon
- esto: oikeuksien tarkistus (palvelinpuolella), deny by default
Lähde: OWASP: A01:2021 – Broken Access Control, 2021, https://owasp.org/Top10/A01_2021-Broken_Access_Control/

2) Find Hidden Web Directories - Fuzz URLs with ffuf
- web-palvelimilla on yleensä piilotettuja hakemistoja, joita ei ole missään näkyvällä sivulla
- ne voidaan löytää automaattisesti kuitenkin "fuzz":aamalla työkalulla "ffuf"
- ffuf on työkalu, joka kokeilee URL-polkuja sanalistasta ja raportoi vastaukset
- "fuzz" saa tehdä vain omissa tai luvallisissa ympäristöissä, koska se on laitonta
- fuzzaamalla saa listan hakemistoista/tiedostoista, joita palvelin vastaa, ja sieltä voi tunnistaa mahdollisia piilotettuja hakemistoja
Lähde: Karvinen, Tero: Find Hidden Web Directories - Fuzz URLs with ffuf, 2023-05-10, https://terokarvinen.com/2023/fuzz-urls-find-hidden-directories/

3) Access control vulnerabilities and privilege escalation
- access control = valvotaan, että kuka saa tehdä ja nähdä mitä web-sovelluksessa
- käyttäjä tunnistetaan ja sitten päätetään mitä hän saa tehdä
- broken access control = käyttöoikeuksia ei valvota oikein: käyttäjä pääsee käsiksi asioihin, joihin hänellä ei ole lupia
- vertikaalinen eskalaatio = käyttäjä pääsee käsiksi esim. adminin toimintoihin
- horisontaalinen eskalaatio = käyttäjä pääsee toisen käyttäjän tietoihin
- Insecure Direct Object Reference (IDOR) = resurssi viitataan suoraan muokattavasta parametrista
Lähde: PortSwigger: Access control vulnerabilities and privilege escalation, https://portswigger.net/web-security/access-control

- väliotsikointi, tarkista kirjoitusvirheet, kirjoita sopivasti tekstiä
- lähdeviittaukset
